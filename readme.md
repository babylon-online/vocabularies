# Vocabularies
The data presented is an export from http://babylon-online.org/vocabularies.

## List of vocabularies
### babylon-pilot-base

Most of the included terms are mapped to [Getty AAT](https://www.getty.edu/research/tools/vocabularies/aat/). The vocabulary information is available as Open Access. (also compare http://babylon-online.org/babylon-projekt/vocabularies/base)
